# 客服回复润色应用设计规范

## 产品定位
极简客服回复润色助手 - 帮助客服人员快速将客户问题转化为专业、规范的回复文本。

## 设计风格选择：柔和卡片风（Soft Card UI）

**核心意象**：如同一位专业、温和的助手在身旁，使用柔和的视觉语言传递专业与友好。

### 配色方案

| 角色 | 色值 | 用途 |
|------|------|------|
| Primary | `#6C63FF` | 主按钮、AI处理状态、关键图标 |
| Primary Light | `rgba(108,99,255,0.12)` | 图标容器背景 |
| Background | `#F0F0F3` | 页面背景 |
| Surface | `#FFFFFF` | 卡片背景 |
| Text Primary | `#2D3436` | 主要文字 |
| Text Secondary | `#636E72` | 次要说明文字 |

| Success | `#00B894` | 成功状态、复制完成 |
| Recording | `#FF6584` | 录音中状态 |

### 交互设计

1. **录音按钮交互**
   - 点击开始 → 图标变为红色 + 脉冲动画
   - 再次点击停止 → 录音图标恢复正常
   - 录音完成 → 自动转文字显示在输入框

2. **截图上传**
   - 点击选择图片 → 底部弹出图片预览
   - 支持删除已上传图片

3. **AI润色流程**
   - 点击"润色"按钮 → 按钮显示loading状态
   - 流式显示润色结果
   - 完成后显示操作按钮（复制/分享）

4. **导出功能**
   - 复制：点击后显示成功提示
   - 分享：调用系统分享面板

### 技术实现要点

1. **语音输入**：expo-av 录制音频 → 上传后端 → 调用 ASR API 转文字
2. **截图上传**：expo-image-picker → FormData 上传到对象存储
3. **AI润色**：后端调用 LLM API 传入图片URL和文本 → 流式返回结果
4. **样式**：使用 Tailwind CSS + 自定义阴影样式

### 组件清单

| 组件 | 状态 |

| PrimaryButton (渐变按钮) | default / loading / disabled |

| ResultCard (结果展示) | empty / loading / has-content |
